# 📊 Capstone – Power BI
File Description: Screenshot of the Power BI executive dashboard created during Capstone Phase 4

## Overview

This folder contains the Power BI report completed as part of the Capstone Project Phase 4. Three MySQL tables were connected to Power BI via Power Query, cleaned, and loaded into a star schema data model with a calendar table built using DAX. Three interactive report pages were developed covering an Executive Overview, Customer Transactions analysis, and Loan Applications analysis.

## Skills Demonstrated
Power Query, DAX Measures, Data Modeling, Calendar Tables, Interactive Dashboards, KPI Cards
