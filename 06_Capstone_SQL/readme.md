# 🗄️ Capstone – SQL
File Description: SQL script containing all analytical queries from Capstone Phase 2

## Overview

This folder contains the SQL script completed as part of the Capstone Project Phase 2. Working with a simulated banking organization's database consisting of 12 tables across 9 source files, analytical queries were written to explore the database structure, uncover insights, and demonstrate proficiency across core SQL concepts.

## Skills Demonstrated
JOINs, GROUP BY, HAVING, Subqueries, CASE Statements, Aggregate Functions
