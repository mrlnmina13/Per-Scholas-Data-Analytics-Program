# 📊 Excel Foundations
File Description: Guided lab practicing logical functions

## Module Overview

This module began with the fundamentals of the Excel interface, covering how to modify columns, rows, and cells, then progressively built into sorting and filtering data and applying conditional formatting. A strong emphasis was placed on understanding the differences between standard, absolute, and structured cell referencing before moving into formulas and functions.

The bulk of the module covered a wide range of functions including logical functions (IF, SUMIF, COUNTIF, AND, OR), lookup functions (VLOOKUP, HLOOKUP, XLOOKUP, INDEX, MATCH), date and time functions, and string manipulation functions such as CONCATENATE, SUBSTITUTE, REPLACE, LEFT, RIGHT, and MID. Statistical analysis was also covered through functions like AVERAGE, MEDIAN, MODE, STDEV, and VAR.

The module wrapped up with data visualization in Excel, exploring combo charts, heatmaps, Pivot Tables, Pivot Charts, and slicers to bring data to life.
