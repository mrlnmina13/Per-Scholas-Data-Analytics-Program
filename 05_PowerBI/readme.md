# 📊 Power BI
File Description: Guided lab building an executive summary report in Power BI

## Module Overview

This module introduced **Power BI Desktop** as a business intelligence tool, covering the full workflow from connecting to data sources to delivering interactive reports. Key topics included data modeling, establishing table relationships, writing **DAX (Data Analysis Expressions)** for custom measures and calculations, building visualizations, and performing analysis to derive actionable insights.
