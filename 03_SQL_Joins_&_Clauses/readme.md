# 🗄️ SQL
File Description: Guided lab practicing SQL joins, clauses, and aggregate functions 

## Module Overview

This module provided a comprehensive introduction to relational database management and SQL, building from foundational concepts to advanced querying techniques.

The module began with the fundamentals of **Relational Database Management Systems (RDBMS)**, covering data modeling and normalization to ensure data is structured efficiently. From there, it moved into **DDL (Data Definition Language)** and data integrity, establishing how databases and tables are created and maintained.

The bulk of the module focused on writing and applying SQL queries, covering SQL clauses, aggregate functions and operators, joins, and subqueries. The module concluded with connecting and importing SQL data into Excel using **ODBC**, bridging database management with spreadsheet analysis.
