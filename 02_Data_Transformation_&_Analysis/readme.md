# 🔄 Data Transformation & Analysis
File Description: Guided lab practicing data transformation with Power Query 

## Module Overview

This module introduced the fundamentals of **ETL (Extract, Transform, Load)** and data transformation, covering various file formats and the tools used to prepare raw data for analysis.

The module began with importing CSV and text files into Excel using Power Query, then progressively built into more advanced techniques such as importing data from the web, performing data cleaning, merging and appending tables, and an introduction to creating a data model using related fields.

Core skills developed in this module include handling missing values, transforming columns, applying calculations, and structuring data efficiently for downstream analysis.
