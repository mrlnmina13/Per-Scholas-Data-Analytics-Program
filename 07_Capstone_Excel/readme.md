# 📊 Capstone – Excel
File Description: Screenshot of the Excel dashboard created during Capstone Phase 3

## Overview

This folder contains the Excel workbook completed as part of the Capstone Project Phase 3. Using Power Query, a live connection was established between Excel and a MySQL database. Data was cleaned, transformed, and loaded into a data model via PowerPivot. A professional summary dashboard was built for leadership along with two additional analysis sheets.

## Skills Demonstrated
Power Query, ETL, Data Modeling, PowerPivot, DAX, PivotTables, Dashboard Design
